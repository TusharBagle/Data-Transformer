# Data Transformer

# 🗄️ Advanced SQL Database Project

## 📌 Project Description
This project is a comprehensive **MySQL relational database system** that demonstrates real-world database design and advanced SQL concepts. It includes multiple related tables such as Customers, Orders, and Employees, and covers a wide range of SQL operations including joins, subqueries, string functions, date functions, aggregation, and window functions.

The goal of this project is to simulate an e-commerce and employee management system while practicing advanced SQL query writing and data analysis.

---

## 🧱 Database Tables

### 1. Customers
Stores customer personal and registration details.

### 2. Orders
Stores order information linked to customers using foreign keys.

### 3. Employees
Stores employee details including department and salary.

## ⚙️ SQL Concepts Covered

### 🔹 Basic Operations
- CREATE TABLE
- INSERT INTO
- SELECT
- UPDATE
- DELETE

### 🔹 Joins
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- FULL OUTER JOIN (simulated using UNION)

### 🔹 Subqueries
- Average-based filtering
- Conditional data extraction

### 🔹 Aggregate Functions
- SUM()
- AVG()

- MIN()

### 🔹 String Functions
- CONCAT()
- UPPER()
- LOWER()
- REPLACE()
- TRIM()

### 🔹 Date Functions
- YEAR()
- MONTH()
- DATE_FORMAT()
- DATEDIFF()

### 🔹 Window Functions
- RANK()
- SUM() OVER() (Running Total)

### 🔹 Conditional Logic
- CASE statements for categorization and discount rules
